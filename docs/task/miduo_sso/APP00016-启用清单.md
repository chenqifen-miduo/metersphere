## 必配环境变量（生产 / CDS）

```bash
MIDUO_SSO_ENABLED=true
MIDUO_SSO_BASE_URL=https://admin.t.ebcone.cn
MIDUO_SSO_APP_CODE=APP00016
MIDUO_SSO_APP_SECRET=<交付单密钥>
MIDUO_SSO_REDIRECT_URI=https://msp.ebcone.net
MIDUO_SSO_SHORTCUT_ID=81
MIDUO_SSO_ORGANIZATION_ID=100001
```

本地也可写入 `local-runtime/conf/metersphere.properties`（目录已 gitignore）：

```properties
miduo.sso.enabled=true
miduo.sso.base-url=https://admin.t.ebcone.cn
miduo.sso.app-code=APP00016
miduo.sso.app-secret=<secret>
miduo.sso.redirect-uri=https://msp.ebcone.net
miduo.sso.shortcut-id=81
miduo.sso.organization-id=100001
miduo.sso.session-ttl-seconds=28800
```

## 启用前门禁

1. 组织 `100001`（或配置的 organization-id）已配置企微通讯录并完成「同步企业微信成员」
2. 存在 `user.wecom_userid` 非空启用用户
3. 部署含 `/auth/miduo/**` 的后端与含落地转发的前端
4. `GET /auth/miduo/status` → `enabled=true` 且 `ready=true` / `reason=OK`
