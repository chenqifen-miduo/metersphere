### 2.1 单模块接入模板（必须遵守）

1. **BE Adapter**  
   - 实现 `ResourceEditAdapter`（`loadPayload` / `applyPayload`）  
   - `resourceType` 写入 `ResourceEditConstants`  
   - 附件 / 关联默认排除  
2. **FE SDK**  
   - 仅用 `useAutoSaveEditor` + `ms-auto-save-status`  
   - 防抖 1.5～2s；保留显式保存与 Ctrl+S  
3. **权限与锁**  
   - 无写权限：`canEdit=false`  
   - 冲突：只读提示，不抢锁  
4. **冒烟**  
   - T1 / T3 / T4 / T7
