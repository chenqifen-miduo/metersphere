# task014 - P2 批量导入幂等

> **预估工期**：1 天  
> **前置依赖**：task010、task003  
> **关联方案**：§15#5、§13 重复导入风险

---

## 1. 任务目标

支持批量导入幂等，避免对话重试或网络抖动产生重复用例。

---

## 2. 任务清单

- [ ] 请求增加可选 `clientRequestId`（或 cases[] 级外部键）  
- [ ] 服务端记录已处理请求与创建的 caseId 映射（表或复用审计扩展——设计评审）  
- [ ] 相同 `clientRequestId` 重放：返回首次 `created`，不新建  
- [ ] 文档约定：Agent 重试必须带同一 id  
- [ ] （可选）MCP `batch_create_functional_cases` 参数透传  

**非目标**：全量用例更新/删除 API（方案一期非目标，仍不在本任务）。

---

## 3. 验收标准

- [ ] 同一 clientRequestId 连续两次 batch-create，平台用例数只增加一次  
- [ ] 不同 id 正常各建一批
