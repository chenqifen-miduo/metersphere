## 3. 任务清单

### 3.1 梳理大数据交互风险

检查：

- 大列表滚动。
- 树节点展开/折叠。
- 节点重命名。
- 拖拽排序。
- 批量选择。
- 跨页选择。
- 筛选后状态恢复。

### 3.2 评估虚拟滚动

按场景评估：

- 普通表格虚拟滚动。
- 树形表格虚拟滚动。
- 左侧模块树懒加载。
- 搜索结果分页。

### 3.3 优化直接 DOM 操作

减少：

- `document.querySelector`
- 手动 dispatch click。
- 依赖第三方组件内部 class。

替代方式：

- ref 控制。
- 组件状态驱动。
- 显式事件。
- 暴露组件方法。

### 3.4 完善加载和失败状态

补充：

- 首次加载。
- 局部加载。
- 分页加载。
- 树节点加载。
- 加载失败重试。
- 空结果。

---

## 4. 验收标准

- [ ] 大列表滚动和操作不卡顿。
- [ ] 树节点操作不依赖脆弱 DOM 选择器。
- [ ] 批量操作和分页状态稳定。
- [ ] 用户能感知加载、失败、重试状态。
- [ ] 优化不改变原业务权限和数据行为。
