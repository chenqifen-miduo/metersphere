## 1. 任务目标

Excel（及必要的 Xmind 对齐）导入时按统一算法拆分步骤，**双写** `steps` + TEXT 全文；**覆盖导入重解析** steps；下载模板在「步骤描述」列写入 **Excel 批注**（不必加回「编辑模式」列）。

---

## 2. 拆分规则（已确认）

| 情形 | 规则 |
|------|------|
| 有序号且无层级（`1.`/`2.` 或 `①`/`②` 或 `【n】`/`[n]` 同级） | 按该序号序列拆成多条 STEP |
| 有序号且有层级 | 只按**最高层级**拆；次级留在同一步正文 |
| 无序号 | **按换行**拆（忽略连续空行） |
| 行中编号 | 仍按有序号规则拆 |

- 步骤列与预期列按段序对齐；不足补空串。  
- 导入后同时生成 TEXT 全文（换行可渲染）。  
- 默认展示类型：可拆则建议 `caseEditType=STEP`，否则 TEXT（仅 UI 默认，数据双写）。
