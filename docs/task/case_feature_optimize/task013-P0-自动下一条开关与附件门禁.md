## 1. 任务目标

用例级四按钮保存成功后，仅在**开关开启**且**有有效附件**且**非末条**时自动下一条；默认关；本机按用户持久化（B1）；开关旁常驻「需有附件才跳」；**测试计划内嵌用例详情同期改**。

---

## 2. 产品规则（已确认）

| 项 | 结论 |
|----|------|
| 默认 | 开关 **关** |
| 触发 | **仅** `handleSetCaseResult` 四按钮；单步结果/汇总改写 **不跳** |
| 附件 | 详情「添加附件」列表；历史附件也算；仅 **done / 已关联**；`init/uploading/error` 不计 |
| 无附件 | 只不跳，**不 Toast** |
| 末条 | 开关开且有附件且无下一条 → `lastCaseTip` |
| 持久化 | **B1** `useStorage('ms.functionalCase.autoNext.{userId}', false)` |
| 显隐 | 与 `showCaseNav` 同显隐 |
| 范围 | 功能用例详情 + **测试计划内嵌** |
