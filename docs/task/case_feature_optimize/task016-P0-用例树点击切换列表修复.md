## 1. 任务目标

点击左侧用例树节点后，右侧用例列表可靠切换为对应模块数据；点树时若处于高级搜索则**退出**高级搜索并按模块刷新；「包含子模块」**保持现网默认**。

---

## 2. 现状分析

| 链路 | 说明 |
|------|------|
| `caseTree` → `caseNodeSelect` | 产出 selectedKeys + offspringIds |
| `index.vue` | 写 `activeFolder` / `offspringIds` |
| `caseTable` `watch(activeFolder)` | 非高级搜索时 `initData` |
| 风险 | 高级搜索跳过模块过滤；`selectedKeys` set 空操作；含子模块 localforage |

---

## 3. 任务清单

### 3.1 复现与定位

- [ ] 记录必现步骤（是否高级搜索、是否关含子模块、叶子/目录）  
- [ ] 确认现网 `getSubShow` 默认值并写入自测说明  

### 3.2 修复

- [ ] 点树时：**退出高级搜索**，按当前 `activeFolder`（+ offspringIds 策略）刷新列表  
- [ ] 修复树选中态与 `activeFolder` 双向一致（`selectedKeys` v-model）  
- [ ] 叶子/目录均强制 `initData`；含子模块逻辑保持现网默认
