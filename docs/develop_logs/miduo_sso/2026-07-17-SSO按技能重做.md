## 重做要点（相对旧实现）

| 项 | 旧问题 | 重做后 |
|---|---|---|
| `X-App-Timestamp` | 误用毫秒 | Unix **秒**（对齐 `OpenSsoRequestVerifier`） |
| 过期字段 | 未认 `sessionExpiresAt` | validate/refresh 优先读 `sessionExpiresAt` |
| 包装响应 | 未校验 `return_code` | `return_code` 为 0/200 后再读 `return_data` |
| 业务成功 | 部分正确 | validate→`valid`；refresh/revoke→`success` |
| 落地页 | token 留在地址栏 | callback 读后立即 strip query |
| 管理登录 | `/#/login/admin` | **保留**账密/LDAP/扫码 |

## 登录入口（不变）

| 路径 | 行为 |
|------|------|
| `/#/login` | 自动跳转米多登录桥；SSO 不可用时不暴露管理员入口 |
| `/#/login/admin` | 账密 / LDAP / 扫码；**仅**手动输入地址进入 |
| `/#/sso/miduo/callback` | QUERY 收 `token`+`state` → POST `/auth/miduo/callback` |

## 核心 API（MeterSphere 侧）

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/auth/miduo/status` | 启用/企微就绪门禁 |
| GET | `/auth/miduo/state` | 生成 CSRF state |
| POST | `/auth/miduo/callback` | `{token,state}` → validate → Shiro Session |
| GET | `/auth/miduo/bridge-url` | 登录桥 URL（含新 state） |
| POST | `/auth/miduo/logout` | revoke + 清 Redis + logout |

## 米多开放接口（后端 HMAC）

- `POST {baseUrl}/api/open/sso/validate-login-token`（signedValue=token）
- `POST {baseUrl}/api/open/sso/refresh-session-token`（signedValue=sessionToken）
- `POST {baseUrl}/api/open/sso/revoke-session-token`（signedValue=sessionToken）
- `GET  {baseUrl}/api/sso/bridge/redirect-url?appCode&redirectUri&state`
