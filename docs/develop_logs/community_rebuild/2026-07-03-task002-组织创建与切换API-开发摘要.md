## 3. 核心流程

### 3.1 创建组织 `add()`

1. 生成 ID、`num`（`ExtOrganizationMapper.getMaxNum()` + 1）  
2. 插入 `organization`  
3. `OrganizationInitService.initOrganization()` — 模板、Bug 状态流  
4. 绑定 `userIds` 为 `org_admin`  
5. 操作日志  

### 3.2 切换组织 `switchOrganization()`

1. 校验 `userId` 与当前登录用户一致  
2. 校验组织存在、用户有权访问（超管跳过成员校验）  
3. `userLoginService.switchUserResource(organizationId, userDTO)`  
4. 返回更新后的 `UserDTO`
