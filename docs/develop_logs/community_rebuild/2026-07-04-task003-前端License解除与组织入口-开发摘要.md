## 3. 核心设计

### 3.1 开关分层

```
VITE_MS_UNLIMITED=true  →  licenseStore.isUnlimited
                              ↓
                         hasLicense() === true
                              ↓
         v-xpack / MsTrialAlert / 资源池 disabled / 组织切换菜单
```

- **未设置** `.env.development.local`：行为与原版社区版一致（`.env.development` 不含 unlimited）
- **设置 unlimited**：前端视为已授权，后端仍依赖 task001 Community Xpack 解除配额

### 3.2 组织切换菜单

原版仅在 `packageType === 'enterprise'` 且 `hasLicense()` 时显示「切换组织」。  
社区版 package 为 `community`，需在 **`hasLicense() && isUnlimited`** 时同样挂载 `switchOrgMenus` 并拉取 `GET /system/organization/switch-option`。
