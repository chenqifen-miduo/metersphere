├── CommunityLicenseServiceImpl.java    # validate() 始终返回 valid
└── CommunityUserXpackServiceImpl.java  # 用户配额校验始终返回 0

- 解除 `NodeResourcePoolService` / `TestResourcePoolService` License 并发限制  
- `GET /license/validate` 返回 `status=valid`

**单元测试**：8 用例全部通过（`mvn test -Dtest=Community*Test`）
