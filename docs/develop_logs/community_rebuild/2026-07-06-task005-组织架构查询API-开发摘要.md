## 1. 本次目标

提供组织架构只读查询 API：通用部门树、管理端部门树（含成员统计）、成员分页、成员详情（脱敏）。

---

## 2. API 清单

| 方法 | 路径 | 权限 |
|------|------|------|
| GET | `/department/tree` | 组织成员可读（登录 + 组织隔离） |
| GET | `/org-structure/departments/tree` | `SYSTEM_ORGANIZATION_PROJECT:READ` 或 `ORGANIZATION_MEMBER:READ` |
| GET | `/org-structure/members/page` | 同上 |
| GET | `/org-structure/members/{id}` | 同上（phone/email/wecomUserid 脱敏） |

---

## 3. 核心实现

- **DepartmentQueryService**：flat list → 内存构树；管理端自底向上聚合 `totalUserCount`
- **OrgStructureMemberService**：PageHelper + `ExtOrgStructureMemberMapper` SQL 分页
- **OrgStructureAccessService**：系统管理员可查任意组织；普通成员仅当前组织
