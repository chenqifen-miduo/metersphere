## 4. 领域与 Mapper

| 类型 | 路径 |
|------|------|
| Domain | `backend/framework/domain/.../domain/Department.java` 等 |
| Mapper | `DepartmentMapper` / `OrgWecomSyncConfigMapper` / `OrgSyncLogMapper` |
| ExtMapper | `backend/services/system-setting/.../ExtDepartmentMapper` |
| User 扩展 | `User.java` / `UserMapper.xml` / `UserExample.java` |

`ExtDepartmentMapper` 提供：

- `listByOrganizationId` — 扁平部门列表（供 task005 构树）
- `selectByWecomDeptId` — 企微部门 ID 换算
- `countEnabledByOrganizationId` — 启用部门统计
